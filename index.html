Bootstrap

Bootstrap is a layout and web UI framework. Focusing on progressive enhancement through media queries, it sets the stage for developers to follow suit and build applications that are suitable for every screen size.

Nearly all css/javascript components are optional and may be removed in custom builds, or modified/extended (most easily through LESS).

It's extremely easy to extend, override, and theme

It provides basic markup for:

  • Layout - Containers, grids, rows with push/pull offsets
  • Forms, tables, content
  • Labels, buttons, navigation, tabs, images, icons, alerts, wells
  • Modal dialogues, image carousels, accordian patterns
  • ... and much more

Bootstrap uses Normalize as its base CSS reset file.

Underscore / Lo-Dash

UnderscoreJS and Lo-Dash are competing libraries of extremely useful javascript utility methods, primarily adding array, function, and object helpers to fill in the gaps.

Underscore is fairly popular and is required for BackboneJS but several in the industry have noted some problems with its interals. Lo-Dash was created to provide the same utility functions (matching the API of underscore), without any of the side effects and much better performance. Read more

Both libraries use the underscore as the class variable. The following example iterates over each element in a collection:

_.each([1, 2, 3], alert);

I'll highlight a few of my personal favorites, but I highly recommend you review the official docs (underscore, lo-dash).

uniq — extracts an array of unique values:

_.uniq([1, 2, 1, 3, 1, 4]);
// => [1, 2, 3, 4]

throttle — limits how often a function is called:

function viewportReize(){
   // js logic for window resizing
}
var thrViewportResize = _.throttle(viewportReize, 50);
$(window).on('resize',thrViewportResize);

debounce — execute after event stops

function repaintPage(){
    // js logic for window resizing
}
var thrRepaintPage = _.debounce(repaintPage, 10);
$(window).on('resize',thrRepaintPage);

extend — combines objects

_.extend({name: 'moe'}, {age: 50});
// => {name: 'moe', age: 50}

clone — shallow-clone an object

// Normally, "b" is a reference to "a"
var a = { name: 'Mike' }
var b = a;
b.name = 'Bob';
console.log(a.name); // Bob

// Cloning splits into two objects
var a = { name: 'Mike' }
var b = _.clone(a);
b.name = 'Bob';
console.log(a.name); // Mike

Underscore also provides a number of type-validation functions like isElement, isArray, isObject, isNumber, isDate and more!
